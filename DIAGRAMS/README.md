## 📋 Deskripsi File

### **File XML Utama:**

1. **USE_CASE_DIAGRAM.xml**
   - Use Case Diagram lengkap sistem
   - Berisi 8 aktor dan 50+ use cases
   - Format XML standar untuk dokumentasi TA

2. **database_diagram_drawio_complete.xml**
   - Diagram database lengkap dalam format DrawIO
   - Menampilkan semua tabel dan relasi

3. **E-BPHTB (Fitur booking online).drawio_edited.xml**
   - Diagram fitur booking online
   - Format DrawIO

4. **E-BPHTB_Swimlane.drawio_edited.xml** & **E-BPHTB_Swimlane.drawio_edited_2.xml**
   - Swimlane diagram proses bisnis
   - Versi 1 dan versi 2 (revisi)

### **File XML Iterasi:**

- **iterasi_1usecase.xml, iterasi_1swimlane.xml, iterasi_1diagram.xml**
  - Diagram untuk Iterasi 1 (Booking hingga Pengiriman)
  - Periode: November 2024 - Januari 2025

- **iterasi_2usecase.xml, iterasi_2swimlane.xml, iterasi_2diagram.xml**
  - Diagram untuk Iterasi 2 (Sertifikat Digital & QR Code)
  - Periode: Februari 2025 - Maret 2025

### **Folder Iterasi_Diagrams:**

Berisi dokumentasi lengkap diagram dalam format Markdown dan XML untuk setiap iterasi:

- **Iterasi 1**: Use Case, Swimlane, Activity Diagram (dalam format MD)
- **Iterasi 2**: Use Case, Swimlane, Activity Diagram (dalam format MD)
- **Iterasi 3**: Use Case, Swimlane, Activity Diagram (dalam format MD dan XML)

### **Folder Gambar:**

- **database_diagram_drawio/**: Screenshot diagram database
- **database_diagram_drawio_complete/**: Screenshot diagram database lengkap

## 🛠️ Cara Menggunakan

### **Membuka File DrawIO:**
1. Buka [draw.io](https://app.diagrams.net/) atau install aplikasi DrawIO
2. File → Open → Pilih file `.xml` atau `.drawio_edited.xml`
3. File akan terbuka dan bisa diedit

### **Membuka File XML Use Case:**
1. Bisa dibuka dengan text editor untuk melihat struktur
2. Bisa digunakan dengan tool UML seperti:
   - Visual Paradigm
   - Enterprise Architect
   - StarUML
   - atau tool UML lainnya yang support XML

### **Membaca Dokumentasi:**
- File `.md` bisa dibuka dengan:
  - Text editor biasa
  - Markdown viewer
  - GitHub/GitLab (jika di-upload)

## 📊 Jenis Diagram yang Tersedia

1. **Use Case Diagram** - Menampilkan aktor dan use case sistem
2. **Swimlane Diagram** - Menampilkan alur proses bisnis per aktor
3. **Activity Diagram** - Menampilkan aktivitas detail dalam proses
4. **Database Diagram** - Menampilkan struktur database dan relasi tabel
5. **Sequence Diagram** - (jika ada) Menampilkan interaksi antar komponen

## 📝 Catatan

- Semua file diagram sudah dipindahkan ke folder ini untuk memudahkan organisasi
- File XML bisa dibuka dengan berbagai tool UML
- File DrawIO bisa dibuka dengan aplikasi DrawIO atau draw.io online
- Dokumentasi dalam format Markdown untuk kemudahan membaca
